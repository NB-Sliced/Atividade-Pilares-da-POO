Herança
A herança é utilizada quando uma classe herda características de outra.
No projeto, a classe Transporte é a base, e as classes Carro, Bicicleta e Aviao herdam dela.

Polimorfismo
O polimorfismo acontece quando classes diferentes possuem o mesmo método, mas com comportamentos diferentes.
No código, todas as classes possuem o método mover(), porém cada uma executa esse método de forma diferente.

Pacotes
O projeto utiliza pacotes para organizar os arquivos.
A pasta transportes contém as classes e possui o arquivo init.py, indicando que ela é um pacote Python.
